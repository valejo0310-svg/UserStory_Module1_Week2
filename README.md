# UserStory_M1_W2
A simple console-based inventory management system built in Python.

## ► Features

°Add products with name, price and quantity.                                                                                                    
°View the full inventory list.                                                                                                                          
°Calculate the total inventory value.                                                                                                           
°Input validation on all fields — no crashes on wrong input.
